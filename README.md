# OFF-projekt
Projektitöö Andmeinseneeria täiendkoolitusprogrammi raames, mis põhineb Open Food Facs andmebaasil.
Open Food Facts (https://af.openfoodfacts.org/discover) on andmebaas, mis koondab toitude pakendiandmeid (triipkood, koostisosad, energia ja toitainete sisaldus, allergeenid, pakendi tüüp ja materjal jne). Andmebaasis on hetkel üle kolme miljoni toote 150 riigist ja vaatahtlikud sisestavad andmebaasi järjepidevalt uusi andmeid. Äriküsimus: kui palju leidub andmebaasis Eestis müüdavaid tooteid, milline on andmete kvaliteet ja millises tempos andmed täienevad.
