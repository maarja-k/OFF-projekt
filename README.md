# OFF-projekt
Projektitöö Andmeinseneeria täiendkoolitusprogrammi raames, mis põhineb Open Food Facts andmebaasil.
Open Food Facts (https://af.openfoodfacts.org/discover) on andmebaas, mis koondab toitude pakendiandmeid (triipkood, koostisosad, energia ja toitainete sisaldus, allergeenid, pakendi tüüp ja materjal jne). Andmebaasis on hetkel üle kolme miljoni toote 150 riigist ja vaatahtlikud sisestavad andmebaasi järjepidevalt uusi andmeid.
**Esialgne äriküsimus: kui palju leidub andmebaasis Eestis müüdavaid tooteid, milline on andmete kvaliteet ja millises tempos andmed täienevad.**
Andmebaasi koopia on alla laaditav (https://world.openfoodfacts.org/data; https://wiki.openfoodfacts.org/Reusing_Open_Food_Facts_Data) ja see uueneb iga päev. Samuti on andmebaasil avalik API (https://openfoodfacts.github.io/openfoodfacts-server/api/).
